# Fundamentos-Python
### Fundamentos de Python aplicados al trabajo con datos

## Descripción del algoritmo
Este proyecto contiene ejercicios prácticos orientados a reforzar los fundamentos del lenguaje Python, con un enfoque inicial en su aplicación al trabajo con datos.

## Funcionamiento básico
Los ejercicios abordan los siguientes temas:

- Creación y uso de variables.
- Operaciones con cadenas de texto.
- Manejo de listas y tuplas.
- Uso de diccionarios.
- Operaciones numéricas básicas.
- Definición y uso de funciones.
- Uso de módulos estándar.
- Control de flujo mediante bucles.

## Resultados y aprendizajes obtenidos
A partir del desarrollo de los ejercicios se logró:

- Reforzar conocimientos previos de Python.
- Aplicar módulos estándar como `math`.
- Comprender la estructura básica de scripts y notebooks.
- Fortalecer buenas prácticas de programación.
